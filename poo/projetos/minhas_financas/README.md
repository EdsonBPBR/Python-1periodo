# Sistema de Finanças

- Adicionar minhas receitas
- Adicionar minhas despesas
- Categorizar minhas receitas e despesas, visando descobrir o principal gasto
- Exibir o saldo total

### Classes
- Transacao
    - Descrição (ex.: salario, bolsa, bonus, 13ª)
    - Valor (R$)
    - Categoria
- Categoria
    - Nome

### Funcionalidades
- Cadastrar transacao (receita / despesa)
- Cadastrar categoria
- Calcular saldo total
